HABILIDADES

Como desarrollador web, poseo habilidades en el uso de diferentes tecnologías y herramientas para crear soluciones digitales de alta calidad. Tengo experiencia en el desarrollo de aplicaciones web utilizando HTML, CSS, JavaScript y PHP, lo que me permite crear sitios web interactivos, dinámicos y atractivos para los usuarios.



Además, tengo habilidades en el control de versiones utilizando Git, lo que me permite colaborar eficientemente con otros desarrolladores y mantener un seguimiento riguroso del progreso del proyecto. Esta herramienta es esencial para garantizar la integridad del código y facilitar su gestión a medida que el proyecto crece y se desarrolla.



También tengo habilidades en diseño de interfaces de usuario utilizando herramientas como Figma y Adobe XD. Entiendo la importancia del diseño UX (experiencia de usuario) y UI (interfaz de usuario), y cómo estas disciplinas son fundamentales para garantizar que los usuarios interactúen de manera efectiva y satisfactoria con las soluciones digitales que desarrollo.
